
# MediOne 💊

MediOne is a **MERN Stack** web application that helps users find nearby medical stores and check the availability of medicines. The platform provides a simple and user-friendly interface to search for medicines, locate pharmacies on an interactive map, and access store information based on the user's current location.

## ✨ Features

* User Registration & Login Authentication
* Search Medicines
* Find Nearby Medical Stores
* Interactive Map with Store Locations
* View Medicine Availability
* Responsive User Interface

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT

## 🌐 APIs & Libraries Used

* **Leaflet.js** – Interactive map rendering
* **Leaflet Routing Machine** – Route and navigation between locations
* **Mapbox** – Map tiles and mapping services
* **Browser Geolocation API** – Retrieves the user's current location

## 🎯 Purpose

MediOne aims to simplify the process of finding medicines by connecting users with nearby pharmacies through a location-based search experience, making medicine discovery faster and more convenient.
